# Ikoner

Vi benytter ikoner som et virkemiddel for hjelpe brukere med å forstå handling eller kontekst. Ikoner kan være med på å skille og fremheve innhold. Det er også et fint navigasjonsverkøy som kan vise varsling av feil eller fortelle brukeren om de forlater nettsiden.

**I Kartverkets designsystem bruker vi ikoner fra [Material Symbols, her finner du den fulle ikonoversikten ](https://fonts.google.com/icons)**

## Eksempler på bruk

```jsx
<span className="material-symbols-outlined">search</span>

// Med farge:
<span className="material-symbols-outlined" style={{ color: "var(--kvib-colors-red-500)" }}>
  warning
</span>
```

Dersom komponenten har støtte for ikon (for eksempel: `icon`, `leftIcon` eller `rightIcon`), så kan du sende inn navnet på ikonet du ønsker på følgende måte:

```jsx
<Button leftIcon="add">Knappetekst</Button>
```
