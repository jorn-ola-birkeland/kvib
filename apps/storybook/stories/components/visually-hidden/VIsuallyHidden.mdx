# Visually Hidden

Visually Hidden er en vanlig teknikk som brukes i netttilgjengelighet for å skjule innhold fra den visuelle klienten, men holde det lesbart for skjermlesere.

## Ta i bruk

```jsx
import { VisuallyHidden, VisuallyHiddenInput } from "@kvib/react";
```
