# Wrap

Wrap er en layout-komponent som brukes til å legge til spacing mellom elements og wrappe dem automatisk hvis det ikke er plass i bredden.

<Canvas of={WrapStories.Wrap} />

## Ta i bruk

```jsx
import { Wrap, WrapItem } from "@kvib/react";
```

**Wrap:** Wrap består av en Box-komponent og rendrer en `<ul>` tag\
**WrapItem:** WrapItem består av en Box-komponent og rendrer en `<li>` tag.
