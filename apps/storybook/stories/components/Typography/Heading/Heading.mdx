# Heading

## Ta i bruk

```jsx
import { Heading } from "@kvib/react";
```

Du kan spesifisere hvilken h-tag headingen skal ha på følgende måte:

```tsx
<Heading as="h2" size="xl">
  Page heading
</Heading>
```

Default er `h2`.
