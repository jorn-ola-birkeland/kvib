# Hooks

Kartverkets designssystem inneholder noen tilpassede hooks som kan gjøre det enklere å håndtere og hente diverse verdier, eksempelvis komponenters states, funksjoner og verider.

## useBoolean

Se [https://chakra-ui.com/docs/hooks/use-boolean](https://chakra-ui.com/docs/hooks/use-boolean) for eksempler og dokumentasjon.

### Ta i bruk

```jsx
import { useBoolean } from "@kvib/react";
```
